# Kapitel 11: Werkzeuge

Auf den Bereich Werkzeuge können Sie über das linke Seitenleisten-Menü zugreifen.

![Seitenleisten-Optionen Werkzeuge.](./assets/learning-ojs3.1-jm-users-tools.png)

Der Bereich Werkzeuge besteht aus einem Reiter Import/Export und einem Reiter Berichtserstellung (Statistik in OJS 3.1.1 und früher).

## Import/Export

Import/Export ermöglicht es Ihnen, einfach Daten aus Ihrer OJS-Zeitschrift zu exportieren und in diese zu importieren.

![Liste von Import/Export-Plugins verfügbar.](./assets/learning-ojs3.1-jm-users-tools-import.png)

Einige der Tools erlauben Ihnen den Export zu Drittanbietersystemen wie PubMed oder DOAJ. Andere ermöglichen den Import oder Export von Benutzer/innen- oder Artikeldaten.

Detaillierte Hinweise zum Importieren und Exportieren finden Sie in der [Administrator/innen-Dokumentation](https://docs.pkp.sfu.ca/admin-guide/en/data-import-and-export).

Weitere Informationen zum Crossref-Export/Registrierungs-Plugin finden Sie im [PKP Crossref Guide](https://docs.pkp.sfu.ca/crossref-ojs-manual/en/).

## Berichtsgenerator

Die Seite Berichtserstellung bietet Zugriff auf eine Vielzahl von Berichten zu Ihrer Zeitschrift.

![Liste der verfügbaren Berichte.](./assets/learning-ojs3.1-jm-users-tools-stats.png)

Das System generiert Berichte mit Daten zur Nutzung der Website und zu den Einreichungen über einen bestimmten Zeitraum. Berichte werden im CSV-Format generiert und können mit einer Tabellenkalkulationsanwendung geöffnet und bearbeitet werden.

- **PKP Nutzungsstatistik-Bericht**: Stellt grundlegende Nutzungsstatistiken für Ihre Zeitschrift bereit.
- **Bericht anzeigen**: Erstellt einen Bericht über die Aufrufe von Fahnen und Abstracts \(d.h. wie oft ein PDF eines Artikels angesehen wurde\).
- **Artikel-Bericht**: Stellt eine Tabelle aller veröffentlichten Artikel zur Verfügung.
- **COUNTER-Bericht**: Stellt COUNTER-Daten bereit.
- **Abonnement-Bericht**: Stellt eine Tabelle der Abonnementaktivitäten zur Verfügung.
- **Begutachtungs-Bericht**: Stellt eine Tabelle aller Begutachtungs-Aktivitäten zur Verfügung.

### Benutzerdefinierten Bericht erzeugen

Nutzen Sie diese Funktion, um eigene Berichte zu erstellen.

![Beispiel für die Einstellungen für einen benutzerdefinierten Bericht zur Erzeugung einer Inhaltsverzeichnis-Vorschau nach Land und Monat für den aktuellen Monat.](./assets/learning-ojs3.1-jm-users-tools-stats-custom.png)

Weitere Informationen zum Generieren und Verwalten von Statistiken in OJS finden Sie in der [Administrator/innen-Dokumentation](https://docs.pkp.sfu.ca/admin-guide/en/statistics).
