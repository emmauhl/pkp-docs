# Creación de contenido accesible: Una guía para editores y autores de revistas

La Guía de Creación de Contenido Accesible cubre los principios generales sobre la creación de contenido accesible universalmente publicado en la web. Destaca las técnicas para abordar limitaciones específicas o el uso de tecnologías de asistencia tales como lectores de pantalla para personas con discapacidad visual o lenguaje de signos para aquellos con limitaciones auditivas. Por último, da consejos específicos sobre la creación de diferentes formatos de galeradas accesibles.

Esta guía será útil para los autores en la preparación de sus manuscritos y para los editores al planear formato y diseño de materiales para la publicación en las revistas.
